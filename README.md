# java
Przykładowe programy w języku Java

1. **FilesInDirectory** - używanie *JFileChooser* i *File*, negocjowanie LaF, tworzenie okienka z guzikiem.
2. **MousePaint** - demo jak można używać metody *paint* do rysowania/malowania i jak przechwytywać zdarzenia generowane przez mysz (czyli do czego służy *MouseListener*, *MouseAdapter* i takie metody jak *mouseClicked*).
